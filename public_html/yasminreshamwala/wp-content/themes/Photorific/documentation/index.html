A) Upload/Install Instructions - top

To install this theme, upload the folder titled Photorific to your /wp-content/themes/ directory. Go into your WordPress admin area and under Appearance -> Themes, select Photorific.

B) General Options - top

When first setting up this theme, you'll need to most likely visit the General options tab. From here, you can enter your logo URL, background image URL and 404 page text.

C) Background Images/Slideshow - top

Setting The Default Background Image (important)

When first setting up your website, you will want to set the default background image. This is the image that will be shown on pages/posts/portfolio items that have no custom background image(s) selected. To set this, in your WordPress admin area, click on the theme options tab on the left sidebar titled 'Photorific'.

From here, click the 'Homepage' tab. In this tab is the option to enter the default background image URL.

Setting Homepage Background Image(s)

To set the background image(s) for your homepage, on the left sidebar in your WordPress admin, there is a tab for the 'Homepage Slider'. To add a new image, simply click 'Add New'.

You can enter any title you want or leave it blank. In the text editor/WYSIWYG enter the URL (only the URL - http://) for the image. Nothing else. If you have any other content in there it will not work so make sure you enter only the URL to the image you want.

Once you've entered the URL, publish and voila, theres your first background image. Of course, to add more just repeat those steps.

Adding Custom Background Images for Posts/Pages

To add a background image for a page, post or portfolio item, simply choose to upload a file from the 'Background Images' section under the text editor when adding a new post/item. After selecting one, you will have an option to add another. You can add as many as you want.

NOTE: When creating a new post/page, there is an option to display the default image or not. If you select yes, this page will just show the one default background image set in the theme options panel.

If you want the custom image/slideshow for that page to show, you will need to select No for the 'Use Default Background Image' option and make sure you have at least one image uploaded.

D) Navigation - top

By default, this theme will automatically add all the pages you've created to the navigation menu. To only include the pages/links you want on the navigation and manage the drop down menu, in the admin area go to Appearance -> Menus.

From here, you can create your own custom menu. Follow these steps to add an item to the navigation menu:

  1. Enter the name for your menu where it says Enter menu name here - it can be anything
  2. Click Create Menu
  3. On the left under Theme Locations, select the menu that you just created and click Save
  4. Now you can start adding items to your menu - to add the first one, select the page/portfolio or gallery category/post that you would like to show on the navigation and select Add to Menu. Once you've done this, again, click Save Menu.
  5. To enter a custom link (other than pages or categories, use the Custom Links section on the left. Enter the URL you would like that link to point to and the title you would like to be displayed under Label.

NOTE: You must click Save Menu after you make changes to the navigation.

Adding Subtext Under Pages For Navigation

To add subtext underneath the title of the link on the navigation, simply cick that links arrow, and edit the Navigation Label. For the subtext, after the main page title, do a forward slash / and the subtext after that.

A few examples:

Home / homepage
About Us / our story
Blog / keep updated
Contact / get in touch

E) Setting Background Images - top

How To Set The Background Image

To enter your background image, go to the theme admin controls page and select the General tab, and find the field titled Background Image URL - In this field, enter the URL of the background image that you want to use for the homepage and for default on pages that don't have their own image set.

Background Images For Posts/Pages

When creating a new post or page, there is an option under the text editor for a background image URL. In that field, enter the URL to the background image you want to be displayed on that specific post/page.

F) Setting Up The Blog - top

This theme utilizes shortcodes to give you more flexibility when creating your blog. To create your blog, create a new page and make sure you've selected the HTML tab, and enter this simple shortcode:

        [blog max="8" meta="true"]

What this will do, is create a blog with pagination that shows only 8 posts per page (you can set it to whatever you want to of course). The meta="true" will just show the post information like who the post was posted by, the date and the comments.

That's it. You can also put your own text or HTML above the shortcode and it will be displayed on each page of the blog.

G) Setting Up The Portfolio & Gallery - top

Portfolio Page

Creating The Portfolio Pages/Categories

To create your portfolio page, navigate to Pages > Add New. You can enter any title/content you want although content is not necessary. Select the Portfolio template from the list under the Page Attributes box and click Publish. This page will display ALL of your portfolio items. If you would like to have different portfolio pages with different categories, you first need to create those categories by following the steps below.

If you only want to display your portfolio page with all portfolio & gallery posts/categories listed together and have no need to create/display multiple categories, you do not need to follow any of the following steps.

------------------------

To create a new category navigate to Portfolio > Portfolio Categories tab.

From here you can create your new portfolio category. Once you've created the category you want, you can then create the items you want to display for that category by following the steps below.

------------------------

IMPORTANT!!!

Once you create a portfolio/gallery category, it is important that you visit the Permalinks page in your admin area. This will make it so when you view a portfolio/gallery category it doesn't return a 404 error. To do this, navigate to Settings > Permalinks tab in your WP admin area. Just by visiting this page, you will refresh the permalinks so your category page does not return a 404 error. You need to do this for BOTH gallery categories and portfolio categories. So, once you create a category for the portfolio and category for the gallery, you will need to visit the permalinks page for each one and that's it.

------------------------

Creating Portfolio Items

To add a new portfolio item, click on the Portfolio > Add New tab in your admin area.

Enter the title you want and the description that you want to display for that portfolio/gallery item in the title and content box. To set the image for your portfolio/gallery item, on the right sidebar click Set Featured Image.

This is the image that will be displayed for this gallery/portfolio item on the portfolio/gallery pages.

Make sure you select the category or categories that you want this item to be displayed in.

Linking To Different Portfolio Categories

If you would like to display a certain category in your navigation, simply navigate to Appearance > Menus tab. If you haven't already, you need to create a custom menu. You can find out how to do that here.

Once you've created the menu, you can add any links that you would like. To display a portfolio or gallery category page, there will be a list of those categories near the bottom of the page that you can easily select and add to the menu.

If you would link to those categories other than using the main theme navigation (like in your pages or anywhere else), you would just need to use the URL of the category. For instance, if you had a category called Photography, and you wanted to link to that category on any pages/posts you create, this would be the code:

        <a href="http://www.mysite.com/portfolio/category-slug">Category Name</a>

To find the slug, navigate to Portfolio Categories or Gallery Categories (whichever you're using) and on that page it will have a table listing all of the categories you've created, their slugs and how many posts are posted in each category. Simply find the category you want and find the slug for that category.

So, if I had say 2 categories named "Web Design" and "Photography", the links would most likely look like (unless otherwise specified):

        <a href="http://www.mysite.com/portfolio/web-design">Web Design</a>
        <a href="http://www.mysite.com/portfolio/photography">Photography</a>

Setting Up The Gallery

To setup the gallery, follow the exact same steps as above but using the Gallery tab instead of the Portfolio tab.

H) Setting Up Templates - top

Contact Page

To setup a contact form, create a new page. Select Contact or Contact Left Sidebar.

This will automatically create a page with an ajax contact form. To setup the email, go into the theme folder photorific. Go into the folder called /js/ and open up contact.php.

In that file near the top, it will say:

		$email = "email@email.com";

Change that to your email address, save it, and upload it to the /js/ folder and that's it.

I) Color Settings - top

This theme comes with a color picker so you can easily select the colors you want to use for just about every part of your website.

Header/Main Sidebar Color Settings

To customize the colors on the header, select the Header/Main Sidebar Color Settings tab and choose the colors from there.

You can easily choose colors for the navigation links and sub text, the logo, and the search background + text color.

Page Color Settings

To change colors for pages, blog posts, headings, the sidebar, etc, select the Page Color Settings tab.

From here, you can change just about every color on the page. You can change the content area background color, text color, heading (h1, h2, h3, etc) colors, link colors and much more.

J) Font Settings - top

From the font settings tab, you can choose your own font sizes along with the font you want to use. This font comes preinstalled with 7 cufon fonts for your use. Easily select the font families you want from the drop down menus.

K) Unlimited Sidebars - top

This theme comes with the sidebar generator plugin already installed giving you the option to create as many sidebars as you want and select a certain sidebar for each individual page.

To create a sidebar, click the Sidebars link under Appearance section in admin area. Click Add Sidebar. Enter the name of your sidebar. Then navigate to the Widgets page and your sidebars will be listed there.

To select a custom sidebar for a page, go to your pages section and select the page you want. Scroll down to the very bottom of the page where it says Sidebars.

It will say:

Replace WP Default Sidebar with None

Where it says None, select the sidebar you want to use, and update the page. That's it!
